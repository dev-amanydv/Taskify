✅ Taskify – Your Personal Task Manager

Taskify is a beautifully simple yet powerful to-do app designed to help you stay organized and boost productivity. Whether you’re managing daily chores, planning a project, or tracking goals, Taskify has all the essential features you need – with a clean interface and optional backend support.

⸻

✨ Features
	•	➕ Add, Edit & Delete Tasks
Create new tasks, update existing ones, or remove them when done.
	•	✅ Mark as Completed / Incomplete
Keep track of your progress with a single click.
	•	🔍 Search Functionality
Quickly find specific tasks using the built-in search bar.
	•	🗂️ Filter Tasks
View tasks by:
	•	All
	•	Active
	•	Completed
	•	📅 Due Dates & Reminders
Never miss a deadline with due dates and reminder options.

⸻

🛠️ Tech Stack
	•	Frontend: React, Tailwind CSS
	•	Backend: MongoDB, Express.js, Node.js
